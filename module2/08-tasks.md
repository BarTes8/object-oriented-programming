## Zadanie 8

Przekształć klasę bazową `Coordinates`, tak aby miała funkcję statyczną

```cpp
static size_t distance(const Coordinates& lhs, const Coordinates& rhs)
```

Funkcja ta powinna zwracać dystans pomiędzy dwoma pozycjami.
